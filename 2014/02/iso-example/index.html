ISO Shutter Speed Example

February 2014 - Tualatin, Oregon

This is a shutter speed example using an X100S in snow at Tualatin Wildlife Refuge on Sunday, February 9th, 2014.

I wanted to get a sense for ISO, aperture, exposure time, and brightness of the resulting image. All photos were taken with +2 EV exposure bias with the mode on auto. I didn’t control for aperture. The results make sense because as the shutter is open longer, more light is captured.

Brightness: 9.75
