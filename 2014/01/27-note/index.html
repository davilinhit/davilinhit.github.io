Oven Control Board v1 »

Monday, January 27 2014

I got super busy with work but then this week have done a ton of rocket club stuff. The GPS RF board is still in design reviews but I think we’ve got the RF side figured out.

I finished an oven control board PCB layout and got it routed for the oven team. All the files are in the Github repo.

We went up to Rocky Butte and shot wifi packets to Council Crest with the DxWiFi team.

I think running around at full speed finally let the cold I’ve been fighting hit me because I was already feeling like crap on Sunday and then I ended up sleeping til the afternoon today. Ugh. Tomorrow will be better.

I just got the email that says my Foxcar control board is in the mail so I guess I know what I’m doing this weekend…
