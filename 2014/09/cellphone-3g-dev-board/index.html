DIY Cellphone 3G Build

September 2014 - Portland, Oregon

David A Mellis designed a cellphone module using the Quectel GSM/GPRS M10 2G module and the Arduino GSM Shield library. The 2G networks in Portland are on their way out so I updated David’s design with his permission and replaced the M10 with a Quectel UMTS/HSDPA UC15 3G module.

The modules have different pinouts, slightly different packages, and the UC15 takes 3.3V-4.3V while the M10 took 3.3V-4.6V, which is still fine for a 3.7V standard LiPo battery.

I got the boards back and did an initial build last week; I’m working with a software dev so we’re going to try to spin it up this week and see where we get.

The currently untested hardware is available for reference at the GitHub repo and released as open hardware under the MIT license. I will update this when it works.

Vocabulary

GSM – Global System for Mobile Communications, 2G
GPRS – General Packet Radio Services, packet data service on 2G and 3G
UMTS – Universal Mobile Telecommunications System, 3G
HSDPA – High-Speed Downlink Packet Access, allows higher speeds 3G speeds**
