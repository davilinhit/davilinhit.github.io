This note is trying out my new script. It first gets the current date in the right format.

NAME=$(date +"%Y-%m-%d")

Then it copies a template I made in _layouts/ and names it using that date output.

cp $SRCDIR/_layouts/note_ .textile $SRCDIR/_posts/$NAME-note.textile

Then it opens it for editing so I can do my thing.

vim $SRCDIR/_posts/$NAME-note.textile

And here it is. Yay for lowering transaction costs.
